# Impostor-clash-royale
Eu estava com dois amigos e tentamos jogar o jogo do impostor que se popularizou nas redes sociais, mas ficou difícil sem mais gente para poder decidir quem seria o impostor, então decidi fazer um código em python que sorteia a carta do clash royale e decide quem começa falando sobre a carta.
Basicamente o sistema tem uma lista completa de cartas do clash royale (posso ter esquecido alguma), randomiza a carta e quem será o impostor por meio de uma pergunta de que quantas pessoas vão jogar, apos ele mostra a carta para os jogadores menos para o impostor ao apertar ENTER pulando cerca de 50 linhas para o proximo jogar não conseguir ver o sorteio do participante anterior, para quem quer brincar com pouca gente ou fazer o grupo inteiro participar coloque o codigo no VS code ou procure no google compilator Python (mais facil) e cole o codigo que esta no arquivo impostor.py, fiz na brincadeira e tedio mas acho que ficou bem legal :)
